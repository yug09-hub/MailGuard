REST API Documentation

Integrate SpamGuard AI into your applications using these endpoints. Base URL: http://localhost:5000

POST /api/predict

Classify a single email with full analysis (spam detection, phishing, URL scan, intelligence).

{
  "email_text": "Your email content here...",
  "model": "nb"  // Options: "nb", "lr", "rf"
}
POST /api/batch_predict

Classify multiple emails at once. Send JSON array or upload CSV.

{
  "emails": ["email1 text...", "email2 text..."],
  "model": "nb"
}
GET /api/history

Retrieve analysis history. Supports query params: filter (spam/ham/all), search, limit.

GET /api/history?filter=spam&search=prize&limit=50
GET /api/stats

Get model statistics, metrics, and available models.

GET /api/analytics

Get analytics data for dashboard charts (distribution, trends, model comparison).

GET /api/models

List available ML models with their performance metrics.

DELETE /api/history/{id}

Delete a specific history entry by ID.
